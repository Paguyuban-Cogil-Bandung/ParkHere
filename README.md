## About ParkHere

APlikasi rekomendasi tempat parkir disekitar dengan keamanan teknologi tinggi serta harga yang sangat transparan:

## Instalasi
- clone repo
- masuk folder, ketik di cmd
- ubah .env.example jadi .env
- `composer install`
- `php artisan key:generate`
- `php artisan serve`
